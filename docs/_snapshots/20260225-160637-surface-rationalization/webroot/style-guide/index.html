Article Content Assembly Map

Publishing contract for article/generic pages: content blocks + optional galleries + optional prototypes.

  • Hero: Optional. Use `layout/body_masthead.ctp` (`.page-hero--single`) when page banner media is present.
  • No banner variant: Use `.article-layout--no-banner` and place page title in body as `.article-body h1`.
  • Body copy: WYSIWYG content in default page body (`.article-body`).
  • Gallery strip: Optional. Insert token in body content only when needed: {{block type="Gallery" id="<id>"}}.
  • Quote callout: Content block or inline HTML wrapper using .article-split .article-callout--quote.
  • Image callout: Inline image wrapper using .article-callout--media .article-media-card.
  • Bottom CTA: Runtime CTA band in `View/Layouts/default.ctp` with Site settings fallbacks.
  • Prototype requirement: none for base article/no-banner layouts; optional `news` prototype for related items module below article body.
