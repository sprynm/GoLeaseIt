# Mode Context Packs

Use these files to load only the context needed for the current function.

## Modes
1. Planning: `docs/modes/planning.md`
2. Design: `docs/modes/design.md`
3. Build: `docs/modes/build.md`
4. Testing: `docs/modes/testing.md`

## Rules
1. Start from `docs/AGENTS.md`.
2. Load one mode pack first.
3. Load `client-information.md` only when client intent is relevant to the task.
4. Add extra docs only when the task requires them.
